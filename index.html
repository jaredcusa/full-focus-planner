Setup Guide
Connect Firebase + GitHub Pages so your planner syncs across every device.

Step 1 — Create a free Firebase project

1
Go to Firebase Console
Visit console.firebase.google.com and sign in with any Google account.
2
Create a new project
Click Add project. Name it full-focus-planner. Disable Google Analytics when prompted. Click through to finish.
3
Register a Web App
Inside your new project, click the </> Web icon. Give it any nickname. You'll see a firebaseConfig object — copy those values, you'll paste them in Step 3.
4
Enable Firestore Database
In the left sidebar go to Build → Firestore Database. Click Create database, choose Start in test mode, pick any region, click Enable. Done.

Step 2 — Host on GitHub Pages (free)

1
Create a GitHub account
Go to github.com and sign up if you don't have one yet.
2
Create a new repository
Click New repository. Name it full-focus-planner. Set to Public. Click Create repository.
3
Upload this file
On the new repo page click Add file → Upload files. Upload this HTML file — rename it index.html first. Commit it.
4
Enable GitHub Pages
Go to Settings → Pages. Under Source select Deploy from a branch, pick main, folder / (root). Save. In a minute your planner will be live at yourusername.github.io/full-focus-planner.

Step 3 — Paste your Firebase config into the file

Open index.html in any text editor (Notepad on Windows, TextEdit on Mac, or VS Code). Find the FIREBASE_CONFIG block near the bottom of the file — it looks like this — and replace the placeholder values with your real ones from Step 1:
// ── PASTE YOUR FIREBASE CONFIG HERE ────────────────── const FIREBASE_CONFIG = { apiKey: "YOUR_API_KEY", authDomain: "YOUR_PROJECT_ID.firebaseapp.com", projectId: "YOUR_PROJECT_ID", storageBucket: "YOUR_PROJECT_ID.appspot.com", messagingSenderId: "YOUR_SENDER_ID", appId: "YOUR_APP_ID" };
Save the file, then re-upload it to GitHub (same steps as before — it replaces the old one). GitHub Pages updates within about 60 seconds.

Step 4 — Open on every device

Bookmark yourusername.github.io/full-focus-planner on your desktop, laptop, and phone. On mobile, add it to your home screen for an app feel:

iPhone/Safari: Tap the Share icon → Add to Home Screen

Android/Chrome: Tap the menu (⋮) → Add to Home Screen

Until Firebase is connected, all your data is saved locally in this browser. It won't be lost — it just won't sync to other devices yet.
